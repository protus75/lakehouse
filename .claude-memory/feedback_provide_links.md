---
name: Always provide web app links
description: Always offer clickable URLs when asking user to check a web app (Streamlit, Dagster, etc.)
type: feedback
---

Always provide the URL/link when asking user to check a web app in the browser.

**Why:** User shouldn't have to remember or ask for the link — it's disruptive.
**How to apply:** Whenever restarting or referencing Streamlit, Dagster, or any web UI, include the full URL (e.g. http://localhost:8501, http://localhost:3000).
